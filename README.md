# Auris Creative

1. Abra esta pasta no Visual Studio Code.
2. Edite o `index.html` para mudar textos, email e Instagram.
3. Abra `index.html` no navegador para ver o site.
4. No GitHub Desktop, crie um repositório chamado `auris-creative`.
5. Use `Repository > Show in Explorer` e copie estes ficheiros para essa pasta.
6. Faça `Commit to main` e depois `Publish repository` ou `Push origin`.
7. No GitHub no navegador, vá a `Settings > Pages`, escolha `Deploy from a branch`, `main` e `/root`.
